# Web デモ実装

Web デモ実装についての仕様を記述する。

# 要件
- ブラウザからアクセスすることを想定する（Web アプリ）
- Web カメラアクセスを利用してカメラ映像を取得する
- カメラ画像を、できる限りリアルタイムに解析するループを作成する
- 解析に対するプロンプトを、ブラウザの画面から設定できる機能を設ける
- 解析結果はログ出力エリアに、解析開始時間を明示して列挙する（ログリスト）
- 使用する VLM のモデルは "LiquidAI/LFM2.5-VL-1.6B" を想定

# 画面構成
画面構成は左側に Web カメラのプレビューを表示し、その右側にログ出力エリアを配置する。
また、カメラビューには「開始」ボタンを配置し、開始ボタンを押したら「停止」ボタンに切り替わり、オン・オフが行えるようにする。
